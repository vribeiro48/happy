<h1 name="run">:construction_worker: How to run the project</h1> <br>

```bash
# Clone Repository
$ git clone https://github.com/vribeiro48/happy.git
```

<h3 name='api'>üì¶ API</h3><br>

```bash
# Go to the 'backend' directory
$ cd backend

# Install the dependencies
$ yarn install

# Start the API
$ yarn dev
```
Access API at http://localhost:3333/

<h3 name='website'>üíª Web</h3><br>

```bash
# Go to the 'web' directory
$ cd web

# Install the dependencies
$ yarn install

# Start React
$ yarn start
```
Go to http://localhost:3000/ to check the website.
